# FIDO2/WebAuthn Management (our-fido2)

`our-fido2` is a comprehensive FIDO2/WebAuthn authentication tool supporting multiple authenticator types including USB, BLE GATT, and Hybrid QR (CTAP2). It provides secure passwordless authentication for ouroborOS systems and web applications.

## Overview

The FIDO2 manager provides:

- **Multi-protocol support** for USB, BLE, and QR authentication
- **Cross-platform compatibility** with FIDO2 standards
- **Secure credential management** with backup and recovery
- **Web application integration** for seamless authentication
- **System integration** with login managers and services

:::note

our-fido2 requires BlueZ with `--experimental` flag enabled for BLE functionality and the Advertising Monitor API. USB and QR authentication work with standard FIDO2 devices.

:::

## Installation

The FIDO2 manager is pre-installed with ouroborOS. Verify installation:

```bash
our-fido2 --version
our-fido2 --help
```

## Basic Usage

### Device Management

```bash
# List connected FIDO2 devices
our-fido2 list

# Show device details
our-fido2 info "My YubiKey"

# Check device capabilities
our-fido2 capabilities "My YubiKey"

# Test device connection
our-fido2 test "My YubiKey"
```

### Credential Management

```bash
# Create new credential
our-fido2 credential create "My Account" --device "My YubiKey"

# List credentials
our-fido2 credential list

# Remove credential
our-fido2 credential remove "My Account"

# Backup credentials
our-fido2 credential backup > credentials-backup.json
```

## Authentication Methods

### WebAuthn Authentication

```bash
# Authenticate with website
our-fido2 auth "example.com" --username "user@example.com"

# Register new account
our-fido2 register "example.com" --username "user@example.com"

# Check authentication status
our-fido2 auth-status "example.com" --username "user@example.com"
```

### System Authentication

```bash
# Add FIDO2 to system authentication
our-fido2 system-auth add

# Remove FIDO2 from system authentication
our-fido2 system-auth remove

# Test system authentication
our-fido2 system-auth test
```

## Advanced Features

### BLE (Bluetooth Low Energy) Support

```bash
# Check BLE readiness
our-fido2 qr-ready

# Start BLE authentication
our-fido2 ble-auth "example.com" --username "user@example.com"

# Monitor BLE connections
our-fido2 ble-monitor

# Configure BLE settings
our-fido2 ble-config --timeout 30
```

### QR Code Authentication

```bash
# Generate QR code for authentication
our-fido2 qr-auth "example.com" --username "user@example.com"

# Scan QR code
our-fido2 qr-scan

# Configure QR settings
our-fido2 qr-config --timeout 60 --size 256
```

### USB Authentication

```bash
# USB authentication
our-fido2 usb-auth "example.com" --username "user@example.com"

# Touch-based authentication
our-fido2 touch-auth "example.com" --username "user@example.com"

# PIN-based authentication
our-fido2 pin-auth "example.com" --username "user@example.com" --pin "123456"
```

## Device Configuration

### PIN Management

```bash
# Set PIN
our-fido2 pin set --new-pin "123456"

# Change PIN
our-fido2 pin change --old-pin "123456" --new-pin "654321"

# Remove PIN
our-fido2 pin remove --pin "654321"

# Reset PIN
our-fido2 pin reset
```

### Biometric Settings

```bash
# Enable biometric authentication
our-fido2 biometric enable

# Disable biometric authentication
our-fido2 biometric disable

# Configure biometric timeout
our-fido2 biometric timeout 30

# Check biometric status
our-fido2 biometric status
```

### Security Settings

```bash
# Enable enterprise attestation
our-fido2 security enterprise-attestation enable

# Configure UV (User Verification)
our-fido2 security uv require

# Configure UP (User Presence)
our-fido2 security up require

# Configure device attestation
our-fido2 security attestation enable
```
