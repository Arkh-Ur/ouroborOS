# Disk Encryption (LUKS + TPM2)

ouroborOS supports optional full-disk encryption via LUKS, with automatic unlocking through TPM2 when available. This provides strong data-at-rest protection without requiring manual passphrase entry on every boot.

## Enabling LUKS

Set `disk.use_luks: true` in your install configuration:

```yaml
disk:
  device: /dev/nvme0n1
  use_luks: true
  btrfs_label: ouroborOS
  swap_type: zram
```

:::caution
LUKS encryption is configured **during installation only**. You cannot enable it retroactively on an existing installation without reinstalling.
:::

## TPM2 Auto-Unlock

When LUKS is enabled and a TPM2 module is present, ouroborOS automatically enrolls the TPM2 chip:

```bash
systemd-cryptenroll --tpm2-pcrs=7+14 /dev/sdX
```

:::note
If no TPM2 module is detected, the system falls back to passphrase-based unlock. You will be prompted for the LUKS passphrase at boot.
:::
