:::note[Inicio Rápido]
Booteá la ISO y el instalador arranca automáticamente. Para installs desatendidos:

```bash
ouroborOS-installer --config /path/to/config.yaml
```

Descargá un template listo para editar desde [install-config.yaml](https://github.com/Arkh-Ur/ouroborOS/blob/dev/templates/install-config.yaml).
:::
