# Hyprland

Hyprland is a modern, dynamic tiling window manager for Wayland that prioritizes simplicity, performance, and customizability. In ouroborOS, Hyprland comes pre-configured with essential applications and a polished user experience, making it perfect for developers, power users, and anyone who values an efficient workflow.

## What is Hyprland?

Hyprland is a tiling window manager that uses the Wayland protocol, offering better performance, security, and stability compared to traditional X11-based window managers. It combines the power of tiling layouts with the flexibility of manual arrangements, all while maintaining excellent performance.

```bash
# Traditional window management - manual
# Install window manager
sudo pacman -S dwm
# Configure manually
# Install terminal
sudo pacman -S alacritty
# Configure applications individually

# Hyprland in ouroborOS - pre-configured
# Hyprland comes with waybar, kitty, rofi, and hyprpaper
# All configurations pre-set for optimal experience
```

## Installation and Setup

### **Hyprland Profile**
Hyprland is included in the "Hyprland" desktop profile during installation:

```bash
# Check if Hyprland is installed
hyprctl --version

# Switch to Hyprland profile (if not already)
ouroboros-profile change hyprland

# Verify Hyprland installation
systemctl status hyprland.service
```

### **Basic Configuration**
Hyprland configuration is stored in `~/.config/hypr/hyprland.conf`:

```bash
# Edit Hyprland configuration
nano ~/.config/hypr/hyprland.conf

# Configuration directory structure
~/.config/hypr/
├── hyprland.conf        # Main configuration
├── hyprpaper.conf       # Wallpaper configuration
├── monitors.conf        # Monitor settings
├── rules.conf          # Window rules
└── apps/               # Application configurations
    ├── kitty.conf
    ├── waybar.conf
    └── rofi.conf
```

## Core Concepts

### **Layouts and Workspaces**
Hyprland supports multiple layouts and workspaces for efficient workflow management:

```ini
# Basic workspace configuration
workspace=1,monitor:HDMI-A-1,layout:dwindle
workspace=2,monitor:HDMI-A-1,layout:master
workspace=3,monitor:HDMI-A-1,layout:flex

# Default layout
monitor=HDMI-A-1,preferred,auto,1
workspace=HDMI-A-1,1
```

### **Window Management**
```ini
# Window rules
windowrule=float, ^(firefox)$
windowrule=size 800 600, ^(vlc)$
windowrule=opacity 0.8, ^(kitty)$

# Window animations
animation=windowsIn,1,3,default,popin 80%
animation=windowsOut,1,3,default,slideoutleft
animation=border,1,3,default
animation=fade,1,3,default
```

### **Keybindings**
Hyprland uses a clean, intuitive keybinding system:

```ini
# Basic keybindings
$mainMod = SUPER

# Window management
bind=$mainMod, Q, killactive,
bind=$mainMod, M, fullscreen,
bind=$mainMod, F, togglefloating,
bind=$mainMod, SPACE, togglesplit,

# Workspace navigation
bind=$mainMod, 1, workspace, 1
bind=$mainMod, 2, workspace, 2
bind=$mainMod, 3, workspace, 3

# Window navigation
bind=$mainMod, H, movefocus, l
bind=$mainMod, L, movefocus, r
bind=$mainMod, K, movefocus, u
bind=$mainMod, J, movefocus, d
```

## Configuration Details

### **Main Configuration File**
```ini
# ~/.config/hypr/hyprland.conf
# -------------------------------
# Monitor and Workspace Setup
# -------------------------------
monitor=HDMI-A-1,preferred,auto,1
monitor=DP-1,preferred,auto,1,transform,1

workspace=HDMI-A-1,1
workspace=HDMI-A-1,2
workspace=HDMI-A-1,3
workspace=HDMI-A-1,4
workspace=DP-1,5
workspace=DP-1,6
workspace=DP-1,7
workspace=DP-1,8

# -------------------------------
# Input Configuration
# -------------------------------
input {
    kb_layout = us
    kb_variant = 
    kb_model = 
    kb_options = 
    kb_rules = 
    
    follow_mouse = 1
    mouse_refocus = 1
    
    touchpad {
        natural_scroll = true
        disable_while_typing = true
        clickfinger_behavior = true
    }
}

# -------------------------------
# General Configuration
# -------------------------------
general {
    gaps_in = 5
    gaps_out = 20
    border_size = 2
    col.active_border = rgba(33,cc,ff,1.0)
    col.inactive_border = rgba(88,88,88,1.0)
    
    layout = dwindle
    # master
    # dwindle
    # master + dwindle
}

# -------------------------------
# Decoration Configuration
# -------------------------------
decoration {
    rounding = 10
    
    blur {
        enabled = true
        size = 8
        passes = 3
        new_optimizations = true
    }
    
    drop_shadow = true
    shadow_range = 4
    shadow_render_power = 3
    shadow_ignore_window = true
    shadow_offset = 0 0
}

# -------------------------------
# Animation Configuration
# -------------------------------
animations {
    enabled = true
    
    bezier = myBezier, 0.05, 0.9, 0.1, 1.05
    
    animation = windows, 1, 3, myBezier
    animation = windowsOut, 1, 3, default, popin 80%
    animation = border, 1, 2, default
    animation = fade, 1, 2, default
    animation = workspaces, 1, 2, default
}
```

## Keybindings and Shortcuts

### **Basic Navigation**
```bash
# Window management
SUPER + Q          # Close focused window
SUPER + SHIFT + Q  # Kill focused window
SUPER + M          # Toggle fullscreen
SUPER + F         # Toggle floating
SUPER + SPACE      # Toggle split mode

# Window focus
SUPER + H          # Focus left
SUPER + L          # Focus right
SUPER + K          # Focus up
SUPER + J          # Focus down

# Window movement
SUPER + SHIFT + H  # Move window left
SUPER + SHIFT + L  # Move window right
SUPER + SHIFT + K  # Move window up
SUPER + SHIFT + J  # Move window down
```

### **Workspace Management**
```bash
# Workspace navigation
SUPER + 1-8        # Switch to workspace 1-8
SUPER + SHIFT + 1-8 # Move focused window to workspace 1-8

# Workspace movement
SUPER + CTRL + H    # Move workspace to left monitor
SUPER + CTRL + L    # Move workspace to right monitor
```

### **Special Applications**
```bash
# Application launchers
SUPER + RETURN      # Open Kitty terminal
SUPER + D          # Open Rofi app launcher
SUPER + SHIFT + D  # Open Rfi window launcher

# System commands
SUPER + SHIFT + E  # Exit Hyprland
SUPER + SHIFT + R  # Reload configuration
SUPER + SHIFT + C  # Clear all windows
```

### **Common Issues**
```bash

systemctl status hyprland.service

# Configuration errors

hyprctl reload              # Reload configuration

# Window focus issues
hyprctl dispatch focuswindow urgent
```

