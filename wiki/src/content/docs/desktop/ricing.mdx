# UI Components & Ricing

Ricing your ouroborOS desktop involves customizing the user interface to create a personalized, efficient workspace that matches your workflow and aesthetic preferences. This guide covers UI components, theme customization, and configuration optimization to help you create the perfect desktop experience.

## What is Ricing?

Ricing refers to the process of extensively customizing a desktop environment to achieve a specific visual and functional style. In ouroborOS, this involves configuring window managers, applications, themes, and utilities to create a cohesive, personalized desktop environment.

```bash
# Default desktop - basic setup
# Limited customization options
# Basic themes and layouts

# Riced desktop - fully customized
# Personal themes and colors
# Custom applications and layouts
# Optimized workflows
# Perfect visual harmony
```

## UI Components Overview

### **Window Manager Components**
```yaml
# Window manager hierarchy
Window Manager → Terminal → Launcher → Panel → Notifications
              ↓           ↓          ↓       ↓          ↓
          Hyprland/Niri → Foot/Fuzzel → Waybar → Dunst → Rofi
```

### **Essential UI Elements**
1. **Window Manager**: Hyprland or Niri with custom configurations
2. **Terminal**: Foot or Kitty with custom colors and fonts
3. **Launcher**: Fuzzel or Rofi with custom themes
4. **Panel**: Waybar or custom status bar
5. **Notifications**: Dunst or custom notification system
6. **Wallpaper**: Hyprpaper or custom wallpaper solution

## Wallpaper and Visual Effects

### **Hyprpaper Configuration**
```bash
# ~/.config/hyprpaper.conf
preload = /path/to/wallpaper.jpg
preload = /path/to/wallpaper2.jpg

wallpaper = HDMI-A-1,/path/to/wallpaper.jpg
wallpaper = DP-1,/path/to/wallpaper2.jpg

ipc = true
```

### **Dynamic Wallpaper Script**
```bash
#!/bin/bash
# ~/.config/scripts/dynamic-wallpaper.sh
# Dynamic wallpaper changing based on time of day

WALLPAPER_DIR="$HOME/wallpapers"

get_wallpaper() {
    hour=$(date +%H)
    
    if [ $hour -ge 6 ] && [ $hour -lt 12 ]; then
        # Morning
        echo "${WALLPAPER_DIR}/morning.jpg"
    elif [ $hour -ge 12 ] && [ $hour -lt 18 ]; then
        # Afternoon
        echo "${WALLPAPER_DIR}/afternoon.jpg"
    elif [ $hour -ge 18 ] && [ $hour -lt 22 ]; then
        # Evening
        echo "${WALLPAPER_DIR}/evening.jpg"
    else
        # Night
        echo "${WALLPAPER_DIR}/night.jpg"
    fi
}

change_wallpaper() {
    wallpaper=$(get_wallpaper)
    hyprpaper -r "${wallpaper}"
    notify-send "Wallpaper changed to $(basename $wallpaper)"
}

# Change wallpaper every 5 minutes
while true; do
    change_wallpaper
    sleep 300
done
```

### **System Integration Script**
```bash
#!/bin/bash
# ~/.config/scripts/theme-sync.sh
# Sync theme across all applications

# Set theme based on time of day
hour=$(date +%H)

if [ $hour -ge 6 ] && [ $hour -lt 18 ]; then
    # Light theme
    gsettings set org.gnome.desktop.interface gtk-theme "Adwaita"
    gsettings set org.gnome.desktop.interface icon-theme "Adwaita"
    gsettings set org.gnome.desktop.interface cursor-theme "Adwaita"
    
    # Update terminal
    sed -i 's/background=.*$/background=ffffff/' ~/.config/foot/foot.ini
    sed -i 's/foreground=.*$/foreground=1a1a1a/' ~/.config/foot/foot.ini
    
    echo "Light theme applied"
else
    # Dark theme
    gsettings set org.gnome.desktop.interface gtk-theme "Adwaita-dark"
    gsettings set org.gnome.desktop.interface icon-theme "Adwaya"
    gsettings set org.gnome.desktop.interface cursor-theme "Adwaita"
    
    # Update terminal
    sed -i 's/background=.*$/background=1a1a1a/' ~/.config/foot/foot.ini
    sed -i 's/foreground=.*$/foreground=e0e0e0/' ~/.config/foot/foot.ini
    
    echo "Dark theme applied"
fi

# Restart applications to apply changes
killall waybar
killall dunst
killall foot

# Restart applications
waybar &
dunst &
foot &
```

## Performance Optimization

### **Window Manager Optimizations**
```ini
# ~/.config/hypr/hyprland.conf
# Performance optimizations
general {
    resize_on_border = true
    no_focus_fallback = true
    gaps_in = 5
    gaps_out = 10
    
    # Disable animations for performance
    animations {
        enabled = false
    }
}

decoration {
    rounding = 0
    blur {
        enabled = false
    }
    drop_shadow = false
}
```


## Configuration Management

### **Configuration Backup**
```bash
#!/bin/bash
# ~/.config/scripts/backup-config.sh
# Backup all custom configurations

BACKUP_DIR="$HOME/config-backups/$(date +%Y%m%d-%H%M%S)"
mkdir -p "$BACKUP_DIR"

cp -r ~/.config/hypr "$BACKUP_DIR/"
cp -r ~/.config/niri "$BACKUP_DIR/"
cp -r ~/.config/foot "$BACKUP_DIR/"
cp -r ~/.config/fuzzel "$BACKUP_DIR/"
cp -r ~/.config/waybar "$BACKUP_DIR/"
cp -r ~/.config/dunst "$BACKUP_DIR/"
cp -r ~/.config/themes "$BACKUP_DIR/"

echo "Configuration backed up to $BACKUP_DIR"
```

### **Configuration Testing**
```bash
#!/bin/bash
# ~/.config/scripts/test-config.sh
# Test all configurations

echo "Testing configurations..."

# Test Hyprland configuration
if hyprctl --config ~/.config/hypr/hyprland.conf &>/dev/null; then
    echo "✓ Hyprland configuration valid"
else
    echo "✗ Hyprland configuration invalid"
fi

# Test Niri configuration
if niri --config ~/.config/niri/config.kdl validate-config &>/dev/null; then
    echo "✓ Niri configuration valid"
else
    echo "✗ Niri configuration invalid"
fi

# Test Foot configuration
if foot --config ~/.config/foot/foot.ini --check &>/dev/null; then
    echo "✓ Foot configuration valid"
else
    echo "✗ Foot configuration invalid"
fi

echo "Configuration testing complete"
```
