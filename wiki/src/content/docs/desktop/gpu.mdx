# GPU Detection and Configuration

orooborOS automatically detects and configures your GPU during installation, providing optimal performance for both 2D desktop and 3D applications. The system supports a wide range of GPU manufacturers and models.

## Automatic GPU Detection

During installation, the system automatically detects your GPU hardware and installs appropriate drivers:

```bash
# GPU detection during installation
lspci | grep -i vga
# Example outputs:
# 00:02.0 VGA compatible controller: Intel Corporation HD Graphics 630
# 01:00.0 VGA compatible controller: NVIDIA Corporation GP107 [GeForce GTX 1050 Ti]
```

:::tip

The GPU detection system also checks for virtualization environments and configures appropriate GPU passthrough for containers and VMs.

:::

- **Radeon Pro**
- **Vega graphics**

```bash
# AMD GPU verification
```

## Driver Installation and Management

### Automatic Driver Installation

Drivers are automatically installed based on GPU detection:

```bash
# View current GPU driver status
our-pac info mesa
our-pac info nvidia-dkms  # If NVIDIA detected
our-pac info xf86-video-amdgpu  # If AMD detected
```

### Manual Driver Management

```bash
# Install specific drivers
our-pac install nvidia-dkms    # NVIDIA proprietary
our-pac install xf86-video-amdgpu  # AMD open source
our-pac install intel-media-driver  # Intel hardware acceleration
our-pac install vulkan-radeon    # AMD Vulkan support
our-pac install vulkan-intel    # Intel Vulkan support
```

## Configuration Options

<Tabs>
<TabItem label="NVIDIA">

### NVIDIA Configuration

```bash
# NVIDIA settings GUI
nvidia-settings

# NVIDIA configuration file
cat > /etc/X11/xorg.conf.d/10-nvidia.conf << 'EOF'
Section "Device"
    Identifier "nvidia"
    Driver "nvidia"
    BusID "PCI:1:0:0"
    Option "DRI" "3"
    Option "NoLogo" "true"
EndSection
EOF
```

```bash
# AMD DRI configuration
```

## Troubleshooting

### Common Issues

```bash
# Check Xorg logs
journalctl -xeu xorg.service

# Verify DRI setup
ls -la /dev/dri/

# Check kernel modules
lsmod | grep -E "(nvidia|amdgpu|i915)"
```

### Driver Reinstallation

```bash
# Reinstall NVIDIA drivers
our-pac -S nvidia-dkms
modprobe -r nvidia
modprobe nvidia

# Reinstall AMD drivers  
our-pac -S xf86-video-amdgpu
modprobe -r amdgpu
modprobe amdgpu
```
