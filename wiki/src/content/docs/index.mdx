:::note[Quick Start]
Boot the ISO and the installer starts automatically. For unattended installs:

```bash
ouroborOS-installer --config /path/to/config.yaml
```

Download a ready-to-edit template from [install-config.yaml](https://github.com/Arkh-Ur/ouroborOS/blob/dev/templates/install-config.yaml).
:::
