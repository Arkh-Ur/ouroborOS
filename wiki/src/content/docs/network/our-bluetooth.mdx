# Bluetooth Management (our-bluetooth)

`our-bluetooth` is a comprehensive Bluetooth management tool built on top of bluetoothctl with experimental Low Energy (LE) support. It provides intuitive commands for device discovery, pairing, and management in ouroborOS.

## Overview

The `our-bluetooth` tool provides:

- **Complete Bluetooth lifecycle management** with 15+ commands
- **Low Energy (LE) support** with experimental features
- **Device discovery and pairing** with automatic authentication
- **Profile management** for various Bluetooth services
- **Audio and peripheral integration**

:::note

our-bluetooth uses bluetoothctl as the backend with experimental extensions for Low Energy functionality. The `--experimental` flag is required for LE features and Advertising Monitor API.

:::

## Installation

The Bluetooth manager is pre-installed with ouroborOS. Verify installation:

```bash
our-bluetooth --version
our-bluetooth --help
```

## Basic Usage

### Device Discovery

```bash
# Start Bluetooth discovery
our-bluetooth scan

# Scan for specific duration
our-bluetooth scan --duration 30

# List nearby devices
our-bluetooth list

# Show device details
our-bluetooth info "My Phone"
```

### Device Management

```bash
# Pair with a device
our-bluetooth pair "My Phone"

# Connect to a device
our-bluetooth connect "My Phone"

# Disconnect from a device
our-bluetooth disconnect "My Phone"

# Remove device
our-bluetooth forget "My Phone"
```

### Status Monitoring

```bash
# Show Bluetooth status
our-bluetooth status

# Show connected devices
our-bluetooth connected

# Show paired devices
our-bluetooth paired

# Show adapter information
our-bluetooth adapter
```

## Advanced Operations

### LE (Low Energy) Features

```bash
# Enable experimental mode
our-bluetooth --experimental le-scan

# LE device discovery
our-bluetooth le-scan

# LE device information
our-bluetooth le-info "BLE Device"

# LE connection management
our-bluetooth le-connect "BLE Device"
```

### Profile Management

```bash
# Show available profiles
our-bluetooth profiles

# Enable specific profile
our-bluetooth profile --enable "A2DP"

# Disable specific profile
our-bluetooth profile --disable "HFP"

# Show profile status
our-bluetooth profile --status "A2DP"
```

### Audio Management

```bash
# Connect audio device
our-bluetooth audio --connect "Headphones"

# Switch audio profile
our-bluetooth audio --profile "A2DP"

# Show audio devices
our-bluetooth audio --list

# Audio quality test
our-bluetooth audio --test
```

## Device Configuration

### Device Settings

```bash
# Set device priority
our-bluetooth priority "My Phone" --level high

# Enable auto-connect
our-bluetooth auto-connect "My Phone" --enable

# Set device trust level
our-bluetooth trust "My Phone" --trusted

# Configure device battery monitoring
our-bluetooth battery "My Phone" --monitor
```

### Security Settings

```bash
# Check device security
our-bluetooth security "My Phone"

# Set device encryption
our-bluetooth encrypt "My Phone" --enable

# Show device authentication
our-bluetooth auth "My Phone"

# Configure device authorization
our-bluetooth authorize "My Phone"
```

## Troubleshooting

### Common Issues

```bash
# Fix connection problems
our-bluetooth fix-connection

# Reset Bluetooth stack
our-bluetooth reset

# Clear pairing cache
our-bluetooth clear-cache

# Restart Bluetooth service
our-bluetooth restart
```

### Diagnostics

```bash
# Run Bluetooth diagnostics
our-bluetooth diagnose

# Check driver status
our-bluetooth driver-status

# Verify hardware
our-bluetooth hardware-test

# Show detailed logs
our-bluetooth logs
```

### Debug Mode

```bash
# Enable debug logging
our-bluetooth --debug scan

# Show detailed device info
our-bluetooth --verbose info "My Phone"

# Monitor connection attempts
our-bluetooth --monitor connect "My Phone"
```
