# Flatpak Wrapper (our-flat)

`our-flat` is a comprehensive Flatpak package wrapper that provides a pacman-style interface for managing Flatpak applications. It integrates seamlessly with ouroborOS, offering familiar commands while leveraging the power and security of Flatpak containers.

## Overview

The `our-flat` tool provides:

- **Pacman-style interface** for familiar Arch Linux users
- **Complete Flatpak lifecycle management** with 8 primary operations
- **Automatic remote management** with curated repositories
- **System integration** with package snapshots and rollback
- **Application sandboxing** with secure containers

:::note

Flatpak applications run in isolated containers, providing enhanced security and preventing system contamination. This makes it ideal for running third-party applications safely.

:::

## Installation

The Flatpak wrapper is pre-installed with ouroborOS. Verify installation:

```bash
our-flat --version
our-flat --help
```

## Basic Usage

### Package Search

```bash
# Search applications in flatpak repositories
our-flat search firefox

# Search with descriptions
our-flat search --desc "video editor"

# Search specific applications
our-flat search code spotify
```

### Package Installation

```bash
# Install a single application
our-flat install firefox

# Install multiple applications
our-flat install code neovim telegram

# Install with specific branch
our-flat install --branch=stable libreoffice
```

### Package Removal

```bash
# Remove application (keep user data)
our-flat remove firefox

# Remove application and user data
our-flat remove --data firefox

# Remove application and all related packages
our-flat remove --recursive firefox
```

## Remote Management

### Configure Remotes

```bash
# List configured remotes
our-flat remotes

# Add Flathub repository
our-flat remote-add flathub https://flathub.org/repo/flathub.flatpakrepo

# Add specific application repository
our-flat remote-add gnome https://sdk.gnome.org/gnome.flatpakrepo

# Remove remote
our-flat remote-remove flathub
```

### Update Remotes

```bash
# Update all remotes
our-flat update

# Update specific remote
our-flat update flathub

# Check remote status
our-flat status flathub
```

## Workflow Management

### Safe Installation Process

```bash
# 1. Create snapshot before installation
our-snapshot create pre-flatpak-install

# 2. Install application
our-flat install code

# 3. Test application
code --version

# 4. If issues arise, rollback
our-rollback now
```

### Update Workflow

```bash
# 1. Check for updates

# 2. Create snapshot

# 3. Update applications
```

## Advanced Flatpak Management

### Remote Configuration

```bash
# List configured remotes
our-flat remote-list
# Output: Name	URL	Enabled
#         flathub	https://dl.flathub.org/repo/flathub.flatpakrepo	yes

# Add Flathub repository (required before installing apps)
sudo our-flat remote-add flathub https://dl.flathub.org/repo/flathub.flatpakrepo

# Add custom repository
sudo our-flat remote-add gnome https://sdk.gnome.org/gnome.flatpakrepo

# Remove remote
sudo our-flat remote-remove flathub

# View remote details
flatpak remote-info --system flathub org.gtk.Gtk3.Theme.Adwaita-dark
```

### System-Wide Installation Policy

```bash
# All installations are system-wide by design
# User installations are not supported in ouroborOS

# Check flatpak installation
flatpak --version

# Verify flatpak is working
our-flat -Ss test

# System-wide install location
ls /var/lib/flatpak/app/
ls /var/lib/flatpak/runtime/
```

### Application Lifecycle Management

```bash
# Check if an app is installed
flatpak list --system | grep org.mozilla.firefox

# Show detailed installation status
flatpak info --system org.mozilla.firefox

# Monitor installation progress
flatpak install --system --noninteractive org.mozilla.firefox
# Shows real-time installation progress
```

### Performance and Cache Management

```bash
# Clean flatpak cache
flatpak cache --system --prune

# Check cache size
du -sh /var/lib/flatpak/cache/

# Update appstream cache for better search results
sudo our-flat remote-add flathub https://dl.flathub.org/repo/flathub.flatpakrepo
# Automatically updates appstream cache
```

### Application Permissions and Security

```bash
# View application permissions
flatpak info --system org.mozilla.firefox | grep "permissions"

# View application sandbox details
flatpak info --system org.mozilla.firefox | grep "sdk"

# Check application isolation
flatpak list --system --columns=application,runtime,origin

# Monitor application resource usage
systemd-cgtop | grep flatpak
```

### Remote Management Best Practices

```bash
# Add trusted remotes only
sudo our-flat remote-add flathub https://dl.flathub.org/repo/flathub.flatpakrepo
sudo our-flat remote-add gnome https://sdk.gnome.org/gnome.flatpakrepo

# Verify remote URLs before adding
curl -I https://dl.flathub.org/repo/flathub.flatpakrepo

# Regular remote maintenance
sudo our-flat remote-remove unused-remote
sudo our-flat remote-list

# Update appstream cache after adding remotes
flatpak update --appstream --system
```

### Backup and Migration

```bash
# Export installed applications
flatpak export --system /path/to/export org.mozilla.firefox org.libreoffice.LibreOffice

# List exportable applications
flatpak list --system

# Import applications on another system
flatpak import /path/to/export

# Application data backup
tar -czf app-backup.tar.gz /var/home/user/.var/app/

# Application data migration
cp -r /var/home/user/.var/app/org.mozilla.firefox /new/home/user/.var/app/
```

### Performance Optimization

